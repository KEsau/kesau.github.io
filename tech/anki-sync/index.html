Anki自定义本地同步服务器
2023-08-25 folder 技术   label 软件使用   阅读次

Anki是一个开源的多平台间隔重复学习软件。

Anki团队提供了免费的Ankiweb给用户在多个客户端之间同步笔记和卡片，然而它速度慢、不稳定、有隐私风险。

这篇知乎文章介绍了一种非常好的本地同步方式。步骤如下：

  1. 进入路由器后台，为PC分配一个静态IP

  2. 新建anki-sync.bat文件，内容如下：

    set SYNC_USER1=user:pass
"C:\Users\YOUR_USER_NAME\AppData\Local\Programs\Anki\anki" --syncserver

  3. 在各个客户端设置"同步-高级设置-自定义同步服务器"，我的例子是：http://192.168.31.126:8080

  4. 执行anki-sync.bat，弹出Windows网络防火墙选择允许公用和私人网络，不要关闭cmd窗口

  5. 在客户端点击"同步"，在弹出窗口中输入用户名user，密码为pass

同步即告成功。
